# Can I reuse an image in future posts?

**Author:** Unknown  
**Created:** 1754299520  
**Updated:** 1754299535  
**Article ID:** 11928104  

---

Yes, you can reuse an image, but it won't happen automatically. To reuse an image that’s already been used, you’ll need to either **manually prompt Soshie in chat** using the image’s exact title or **re-upload the image** so it's treated as new by the system.
